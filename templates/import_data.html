♻️ Restoring from a Backup?

If you downloaded a backup from 💾 Download Full Backup, here is exactly how to restore:

  1. Open the backup .xlsx file — it has 5 sheets: Stock, Purchases, Sales, Due Payments, Trade Dues.
  2. Restore in this exact order to avoid errors: Purchases → Stock → Sales → Due Payments → Trade Dues.
  3. For each sheet: go to that sheet in Excel, save it as a new .xlsx file, then upload it below with the matching Import Type.
  4. Do not rename or delete any column headers — the system reads them exactly.
  5. You can also upload the full backup file directly — the system will auto-detect the header row.

💡 Tip: If a row fails to import, it will be listed below as an error and skipped — other rows are not affected.
